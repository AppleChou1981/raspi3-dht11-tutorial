DHT11 溫溼度數據收集：Google Apps Script 快速部署方案 (方案 A)

👉 切換至【方案 B：GCP API 本地中繼與直連方案】
⚠️ 重要提醒：請先插妥杜邦線並確認所有接線正確後再行送電！若是電源（VCC）與接地（GND）接反，會導致 DHT11 及 Raspberry Pi 晶片過熱燒毀！
💡 架構導讀： 本方案由樹莓派定時讀取 DHT11 後，發送加密 JSON 到 Google Apps Script (GAS)。GAS 負責將數據寫入試算表，並由 GAS 後台直接呼叫 LINE API 發送通知，實現不佔用樹莓派效能的雲端中轉架構。
第一步：取得 LINE 通訊憑證 (金鑰)

在使用程式碼前，您必須先從 LINE Developers 取得發送權限：

1. 取得 Your user ID：
登入 LINE Developers ➡️ 點進您的 Channel ➡️ Basic settings 頁籤 ➡️ 滑至最底部複製 Your user ID (U開頭的超長亂碼)。
2. 取得 Access Token：
切換至 Messaging API 頁籤 ➡️ 滑至最底部 ➡️ 在 Channel access token 欄位點擊「Issue」並複製該金鑰。

⚠️ 關鍵動作：請務必用手機掃描 Messaging API 頁面之 QR Code 將您的機器人**加入好友**，否則系統將無法成功推送訊息。

第二步：如何檢索並提取 Google 試算表 ID？

打開您的雲端試算表，觀察瀏覽器上方網址列，結構如下圖。請僅複製兩條斜線中間的那一長串亂碼：

https://docs.google.com/spreadsheets/d/【 只要複製這整串隨機亂碼，這就是試算表 ID 】/edit#gid=0

💡 註：這串 ID 稍後必須貼進 Python 與 GAS 程式碼中的欄位。

第三步：硬體腳位與型號提醒

⚠️ 型號與電阻關鍵確認：

  • 3 腳位模組（常見藍色小板）： 通常已內建上拉電阻，不需外接，直接連接即可。
  • 4 腳位元件（裸件）： 內部無整合電阻，必須在 VCC 與 DATA 之間手動並聯一個 4.7kΩ ~ 10kΩ 的上拉電阻。

第四步：雲端後台 Google Apps Script 指令碼

請將以下代碼貼入試算表的 Apps Script 編輯器，並代換您的 LINE 憑證：

function doPost(e) {
  var params = JSON.parse(e.postData.contents);
  var targetId = params.spreadsheet_id; 
  var temp = params.temperature;
  var humi = params.humidity;
  
  // 1. 寫入歷史紀錄
  var ss = SpreadsheetApp.openById(targetId);
  var sheet = ss.getSheets()[0];
  var now = Utilities.formatDate(new Date(), "Asia/Taipei", "yyyy-MM-dd HH:mm:ss");
  sheet.appendRow([now, temp, humi]);
  
  // 2. LINE Bot 設定 (在此代換第一步取得的憑證)
  var LINE_ACCESS_TOKEN = "請貼上您的_CHANNEL_ACCESS_TOKEN";
  var TARGET_USER_ID = "請貼上您的_YOUR_USER_ID";
  
  var line_api_url = "https://api.line.me/v2/bot/message/push";
  var message_text = "\n🌡️ 樹莓派雲端 GAS 回報\n-------------------\n⏰ 時間：" + now + "\n°C 溫度：" + temp + " °C\n💧 濕度：" + humi + " %";
  
  var options = {
    "method": "post",
    "headers": {
      "Content-Type": "application/json",
      "Authorization": "Bearer " + LINE_ACCESS_TOKEN
    },
    "payload": JSON.stringify({
      "to": TARGET_USER_ID,
      "messages": [{"type": "text", "text": message_text}]
    }),
    "muteHttpExceptions": true
  };
  
  UrlFetchApp.fetch(line_api_url, options);
  return ContentService.createTextOutput("Success: Logged to Sheet and Sent to LINE.");
}
第五步：樹莓派用戶端 Python 傳送程式碼
import time
import requests
import board
import adafruit_dht

# 📢 填入您的參數
GAS_WEBAPP_URL = "https://script.google.com/macros/s/XXXXX/exec"
SPREADSHEET_ID = "您的Google試算表ID"

# 連接實體第 7 針腳 (GPIO 4)
dhtDevice = adafruit_dht.DHT11(board.D4)

print("樹莓派 方案 A 系統啟動，每 10 分鐘自動採樣一次...")

try:
    while True:
        try:
            temperature = dhtDevice.temperature
            humidity = dhtDevice.humidity
            
            if humidity is not None and temperature is not None:
                payload = {
                    "spreadsheet_id": SPREADSHEET_ID,
                    "temperature": round(temperature, 1), 
                    "humidity": round(humidity, 1)
                }
                res = requests.post(GAS_WEBAPP_URL, json=payload, timeout=15)
                print(f"【方案 A】上傳成功！回應: {res.text}")
                
            time.sleep(600) 
        except RuntimeError as error:
            # 物理通訊偶發錯誤，等待 2 秒重試
            time.sleep(2.0)
            continue
        except Exception as error:
            dhtDevice.exit()
            raise error
except KeyboardInterrupt:
    dhtDevice.exit()
