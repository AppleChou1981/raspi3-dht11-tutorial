本專案為物聯網進階實作之「方案 A（GAS 快速部署方案）」。系統以樹莓派 3 為核心，透過 Python 定時讀取 DHT11 溫溼度感測器數據。本方案採用輕量化的網頁通訊架構，樹莓派端僅需將環境數據封裝為 HTTP POST 請求，主動發送至部署好的 Google Apps Script (GAS) 網頁應用程式網址（Web App URL）。由雲端 GAS 指令碼扮演中央控制角色，自動解析 JSON 欄位並追加寫入雲端試算表，同時向後端串接 LINE Messaging API（或 LINE Notify），將即時環境數據字卡流暢推播至使用者手機。此方案特別適合講求本機端程式碼精簡、快速佈署與跨平台整合的基礎物聯網應用。

This project demonstrates "Plan A (GAS Rapid Deployment Solution)" for an advanced IoT implementation. Powered by Raspberry Pi 3, the system uses Python to periodically read temperature and humidity data from a DHT11 sensor. This approach leverages a lightweight web communication architecture: the Raspberry Pi simply encapsulates environmental data into an HTTP POST request and sends it to a deployed Google Apps Script (GAS) Web App URL. The cloud-based GAS script acts as a central controller, automatically parsing the JSON payload to append rows to Google Sheets, while simultaneously interfacing with the LINE Messaging API (or LINE Notify) to push real-time data cards to the user's mobile device. This solution is ideal for IoT applications that prioritize clean client-side code, rapid deployment, and seamless cloud-to-mobile integration.
